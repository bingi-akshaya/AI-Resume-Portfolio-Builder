# AI Resume & Portfolio Builder

This project is a web application that automatically generates resumes and portfolios using AI.

## Features
- AI Generated Resume Templates
- Portfolio Website Generator
- Download Resume as PDF
- Easy Customization
- Instant Sharing

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning

## How to Run

1. Install requirements
pip install -r requirements.txt

2. Run the application
python app.py
